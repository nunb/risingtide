# Rising Tide

Rising Tide is the Copious service for processing feed stories.

## Project Setup

# `git clone git@github.com:utahstreetlabs/risingtide.git`
# `cd risingtide`
# `git submodule init`
# `git submodule update`
# `cd actions-solr && make`
# `gem install bundler`
# `bundle install`
# `foreman start`

### Tests

# `bin/run-test-solr`
# `lein midje`
